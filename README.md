# Scheduler tasks
A simplified api to call/schedule tasks in windows using ``schtasks``

```
say -h
usage: say [-h] [-i INPUT] [text [text ...]]

Say in windows

positional arguments:
  text                  sentence to speak

optional arguments:
  -h, --help            show this help message and exit
  -i INPUT, --input INPUT
                        Text File to speak
```
